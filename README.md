## 香港 CN2 VPS 是什么，为什么国内用户会专门找它

**定义先说一遍**：香港 CN2 VPS，指的是机房位于香港、回程走中国电信 CN2 GIA（AS4809）优化线路的虚拟专用服务器。CN2 GIA 是中国电信旗下专门针对国际互联的高端网络，和普通 BGP 相比，最大的区别是拥塞更少，晚高峰不容易掉速。

香港机房对国内用户有两个核心优势：

一是物理距离近。从大陆打过去的延迟，Premium 线路通常在 20 到 50ms 之间，洛杉矶的优化线路怎么跑也得 150ms 起步。延迟直接影响网页首字节时间（TTFB），这块香港没得说。

二是走 CN2 GIA 的话，线路经过的节点相对固定，不像普通国际出口可能绕日本、绕美国再回来，路由稳定性好很多。

不过有一点要说清楚：香港 VPS 不等于都有 CN2 优化。有些机房虽然在香港，但去程和回程走的是 NTT 或者普通 Tier 1 线路，对国内用户来说延迟和稳定性跟美国西海岸差距不大，甚至更差。所以选香港机房，线路比位置更重要。

## 常见问题

**Q：香港 CN2 VPS 和普通香港 BGP VPS 有什么区别？**

CN2 GIA 是中国电信的精品网络，和普通 BGP 相比，回程绕路少，晚高峰限速概率低。BGP 便宜，多运营商接入，但拥塞的时候谁都顾不上。如果你的业务靠大陆用户访问，CN2 线路值得花那个溢价。

**Q：DMIT 香港 Premium 适合哪些人？**

电信或联通用户为主的建站需求最合适，延迟低，稳定性好。如果你只是偶尔需要香港 IP 做测试，或者用户不在国内，$39.90/月 买 TINY 有点浪费——Eyeball 或 T1 完全够。

**Q：香港 Eyeball 和 Premium 差距大吗？**

Eyeball 走 CMI 优化，不是 CN2 GIA。两者在白天差距不明显，晚高峰时 Premium 更稳。如果你对延迟不是特别敏感，Eyeball 是更平衡的选择；如果你在意那 10-20ms 的差距、晚高峰也要稳，那就上 Pro。

**Q：流量超了会停机吗？**

Pro 和 Eyeball 系列超量后通常限速处理，不会直接停机。T1 系列超量后限速 50Mbps 继续运行。具体政策以官网为准，购买前建议在产品页确认一下。
